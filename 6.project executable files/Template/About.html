Airbnb Price Forecasting

Unveiling the Patterns in Airbnb Pricing

About Airbnb

Airbnb is an online marketplace that connects people looking to rent out their homes with those looking for accommodations. Founded in 2008, Airbnb offers a variety of properties ranging from single rooms to entire homes and unique stays like treehouses and castles.

Project Overview

This project aims to forecast Airbnb prices using machine learning models. By analyzing various factors such as location, property type, amenities, and reviews, we can predict future rental prices with high accuracy.

Model Results

Model MAE RMSE
Linear Regression 120.34 150.67
Random Forest 95.23 130.45
